# 서울 지하철 라이브

서울 지하철 1~9호선 열차가 지금 어디쯤 오고 있는지 지도 위에서 실시간으로 보여주는 단일 파일 웹앱입니다.

실시간 API는 열차의 현재 역과 상태(진입, 도착, 출발)만 제공합니다. 이 앱은 노선을 그래프로 내장하고 행선지 방향을 역산해서, 열차를 역과 역 사이 구간에 보간 배치합니다. 갱신할 때마다 마커가 부드럽게 이동합니다.

## 주요 기능

- 1~9호선 전체 지원 (분기, 순환, 지선 포함 404개 역 내장)
- 기본(다크), 구글, 네이버 3가지 지도 전환
- 급행과 막차 표시, 상행/하행(내선/외선) 필터
- 인증키 없이도 동작하는 모의 데이터 모드
- 갱신 주기 선택(15/30/60초)과 일일 1,000회 호출 한도 관리용 카운터

## 실행 방법

1. 서울 열린데이터광장(data.seoul.go.kr)에서 **실시간 지하철 인증키**를 발급받습니다. 일반 인증키가 아닌 실시간 전용 키여야 합니다.
2. `config.example.js` 를 `config.js` 로 복사하고 키를 입력합니다.
3. 로컬 서버로 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 으로 접속하면 됩니다. 키가 없으면 모의 데이터 모드로 시작합니다.

## 지도 전환 (선택)

설정에서 구글 또는 네이버 지도로 바꿀 수 있습니다.

- 구글: Google Cloud Console에서 Maps JavaScript API 키 발급 후 `config.js` 에 입력
- 네이버: 네이버 클라우드 플랫폼에서 Maps 이용 신청 후 클라이언트 ID 발급. 콘솔의 Web 서비스 URL에 `http://localhost:8000` 을 등록해야 인증이 통과됩니다.

## 참고

- 열차 위치는 역 단위 상태를 보간한 추정값으로, 실제 위치와 차이가 있을 수 있습니다.
- 최근 개통 구간 일부 역(진접선, 하남선, 별내선, 9호선 3단계)은 근사 좌표를 사용합니다.
- 데이터 출처: 서울 열린데이터광장 지하철 실시간 열차 위치정보 API
